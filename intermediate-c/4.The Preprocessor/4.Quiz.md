## Fill in the blanks to create a function that takes an integer as its parameter, calculates, and returns the sum of all whole numbers from 1 to that number.

```cpp
int calc(int n) {
    int result = 0;
    
    for (int i = 1; i <= n; i++) {
        result += i;
    }
    
    return result;
}
```
